# Калькулятор на Python

Простое консольное приложение калькулятора с тестированием.

## Как использовать

1. **Скачайте файлы** (нажмите Code → Download ZIP)
2. **Распакуйте архив** на компьютере
3. **Запустите калькулятор**:
   - Откройте терминал в папке проекта
   - Выполните: `python src/main.py`

## Функции калькулятора

- Сложение (+), вычитание (-), умножение (*), деление (/)
- Возведение в степень (^)
- Факториал (!)
- История операций
- Обработка ошибок
